初识Linux

第一次接触Linux的原因是由于笔记本需要安装固态硬盘，我把固态设置成了C盘。为了整齐美观，就重装了一下系统。由于之前的Windows10实在是卡顿，所以就借机安装了Windows10和Ubuntu双系统。

最初安装Ubuntu时，上网查找了一些教程，发现不是特别难。只是要给各个分区划分大小比较困惑。

还有就是安装时不要联网，因为联网后，会在安装时自动联网下载一些文件。由于某种原因，国内的下载速度时很慢的，这会延长安装的等待时间。

第一次安装

安装过程，异常顺利。但是我删除了语言设置里的其他语言选项，导致Ubuntu不能开机。无奈只好将磁盘全部清除，重新安装。

第二次安装

有了上次的经验，这次顺风顺水。不过也由于第一次的失误，第二次安装完毕弄到了凌晨2点多。

第一次使用Ubuntu

Ubuntu的默认桌面是“仿生海狸”，简洁的线条加上舒适的配色。第一眼看到，我就非常喜欢。

再看看状态栏和设置界面，很像Android的风格。这样说或许不太对，是Android很像Ubuntu的风格。


加深了解

每每遇到一个新设备或新系统，我最先要做的就是两件事。

1.打开设置界面，看看都有哪些设置选项。体验新设备或系统的设置选项很有意思。

2.打开文件管理系统，文件管理系统能看到当前系统都有什么文件，我对文件的格式很敏感。

文件管理界面

文件管理界面的打开方式和Window有很大不同。Ubuntu的文件管理图标只需单机一次就可以打开。而Windows的打开方式是双击（或许也可以设置成单机一次就打开，但我这里说的是默认的设置）

第一次打开就双击的我很是不习惯这种设置，几乎每次都会双击，打开两个文件管理器。后来就慢慢习惯了。

经常使用的命令

命令 作用
cd 切换目录
ls 列出当前目录里的文件和目录
mkdir 创建空文件夹
rmdir 删除空文件夹
chmod 修改文件夹或文件的权限
pwd 显示当前所在目录
cp 复制文件到某文件夹
rm 删除文件或文件夹
mv 移动文件或文件夹
cat 从文件第一行显示内容
tac 从文件最后一行开始显示内容
df 显示磁盘占用情况
tab键 tab键补全命令非常方便
init 0 关机
shutdown 关机，可配合其他选项使用
reboot 重启，可配合其他选项使用
